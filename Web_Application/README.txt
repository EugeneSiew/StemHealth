2. app.py - The script to start the web application


*Steps to start* 

1. Install all the libraries and make sure python is installed.

2. Open a terminal in this directory (or the same directory in where the app.py resides).

3. Enter the command "python app.py"

4. Wait for the confirmation message to show, then visit http://127.0.0.1:5000 or http://localhost:5000/ to view the web application
